# Appendix C: Project Risk Register

This appendix contains the project-delivery risk register.

Current files:

- `Project Risk Register - B-BOT.md`
- `Project Risk Register - B-BOT filled.docx`
- PDF companion export: `../pdf_exports/Appendix_C_Project_Risk_Register_companion.pdf`

This appendix is separate from Appendix E. Appendix C covers risks to successful project delivery, such as hardware delay, unstable tuning, data availability, scope control, repository evidence and final report pressure. Appendix E covers health and safety hazards to the student or other people during practical work.
